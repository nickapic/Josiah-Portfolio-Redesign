# Photographer Portfolio Template

A photography portfolio site template.Made with html,scss,js

to edit the templates css all you have to do is to just download node js then in the terminal traverse to this file and do npm install which will download all the needed dependencies.

Then all you have to do is write npm run watch:sass in your terminal and then you can edit the scss files and everytime you save it the changes will be made live and with the help of live sever packae which is already installed you can lie previewthose changes.
